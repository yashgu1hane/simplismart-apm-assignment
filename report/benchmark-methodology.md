# Benchmark Methodology

## Objective

Compare Simplismart and Fireworks AI using the same LLM model (Llama 3.1 8B).

## Prompt Dataset

* 20 prompts
* 5 categories

  * Knowledge
  * Summarization
  * Writing
  * Product
  * Reasoning

## Benchmark Approach

### Validation Run

* 20 requests per platform

### Final Run

* 100 requests per platform

## Metrics Collected

* Response Latency (ms)
* Success Rate
* Response Length
* Category-wise Performance

## Output Format

Results stored in CSV for analysis and comparison.
